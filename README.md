# Securities Litigation Survival Analysis

Replication package for:

**"The Litigation Clock: Competing Risks and the Duration of U.S. Securities Class Actions"**
Andrew Yang — Princeton University, ORFE Senior Thesis, 2026

---

## Overview

This project models the time-to-resolution and outcome type (settlement vs. dismissal) of U.S. federal securities class action lawsuits filed between 1990 and 2024. Using competing-risks survival analysis, we estimate how the Private Securities Litigation Reform Act (PSLRA, 1995), federal circuit, and case characteristics jointly shape both the speed and type of resolution.

**Core question:** How did the PSLRA reshape the litigation clock — and did its effect on settlement and dismissal hazards persist after accounting for concurrent changes in case composition?

**Methods (in pipeline order):**
1. Kaplan-Meier / Nelson-Aalen estimators
2. Cumulative Incidence Functions (Aalen-Johansen, competing risks)
3. Cause-specific Cox proportional hazards models
4. Fine-Gray subdistribution hazard models
5. IPTW-weighted marginal structural models (composition-adjusted, not causal)
6. Shared frailty models (circuit-level random effects)
7. Model diagnostics (C-index, time-dependent AUC, Schoenfeld residuals, covariate balance)
8. Robustness checks (alternate disposition coding schemes B and C)

---

## Data

**Source:** [Federal Judicial Center (FJC) Integrated Database (IDB)](https://www.fjc.gov/research/idb)

The raw data file is `cv88on.txt` (Civil cases, 1988–present), available for free download from the FJC website. The universe of analysis is NOS code 850 (Securities class actions), yielding approximately 12,968 cases.

**The raw data and cleaned intermediates are not committed to this repository** (files are large; see `.gitignore`). To replicate, download the FJC IDB civil file and place it at:

```
data/raw/cv88on.txt
```

Then run the pipeline in order (see below) to regenerate all cleaned data and model outputs.

## Replication

### Requirements

R version 4.3 or later. Install required packages:

```r
install.packages(c(
  "tidyverse", "lubridate", "janitor", "here",
  "survival", "cmprsk", "tidycmprsk", "ggsurvfit",
  "coxme", "WeightIt", "cobalt",
  "timeROC", "pec",
  "kableExtra", "xtable"
))
```

### Running the Pipeline

Run all scripts from the project root in order:

```bash
Rscript code/01_clean.R          # ~2 min — builds cleaned .rds files
Rscript code/02_descriptives.R   # KM curves, CIF figures, summary tables
Rscript code/03_cox_models.R     # Cox models; saves output/models/cox_models.rds
Rscript code/04_fine_gray.R      # Fine-Gray models; saves fine_gray_models.rds
Rscript code/05_propensity_scores.R  # PS overlap figure and IPTW diagnostics
Rscript code/05_causal_iptw.R    # IPTW weighted models; saves iptw_results.rds
Rscript code/06_frailty.R        # Frailty models; saves frailty_results.rds
Rscript code/07_diagnostics.R    # C-index, AUC, Schoenfeld; requires prior .rds files
Rscript code/08_robustness.R     # Scheme B/C robustness; saves robustness_results.rds
```

Each script is independently runnable and writes its outputs to `output/figures/`, `output/tables/`, and `output/models/`.

### Disposition Coding Schemes

Three competing-risks coding schemes are implemented in `01_clean.R`:

| Scheme | Settlement | Dismissal | Notes |
|--------|-----------|-----------|-------|
| **A (primary)** | Code 13 + judgment codes with `JUDGMENT=1` | Codes 2,3,12,14 + judgment codes with `JUDGMENT=2` | Ambiguous `JUDGMENT` values censored |
| **B (liberal)** | Scheme A + code 12 reclassified as settlement | Same as A otherwise | Voluntary dismissals treated as settlements |
| **C (expanded)** | Scheme B + code 5 reclassified as settlement | Same as B otherwise | Consent judgments added to settlements |

All main results use Scheme A. Schemes B and C are evaluated in `08_robustness.R`.

## Key Results

| Model | PSLRA Settlement HR | PSLRA Dismissal HR |
|-------|--------------------|--------------------|
| Baseline Cox | 0.563 | 1.409 |
| Extended Cox | 0.784 | 1.661 |
| Fine-Gray (SHR) | 0.503 | 1.719 |
| IPTW-weighted Cox | 0.741 | 1.519 |

Sample (Scheme A): 3,801 settlements / 5,971 dismissals / 3,196 censored (n = 12,968).
